This is a solution for RiskIdent to demonstrate the working deployment of API in a Kubernetes environment using Helm charts.The solution allows you to deploy an API in a zero downtime 
fashion with minimum of three nodes running in a production environment.

Prerequisites

1: Docker.
2: Kubernetes.
3: Helm chart.
4: Kubectl


Following are the steps to run the soultion:

1: Clone the repository 
2: Open Command Prompt and enter helm install . --generate-name
